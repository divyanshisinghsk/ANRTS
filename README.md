# ANRTS
Aadhi Nadi Water Tracking System is a student project tracking various parameters in waterbodies and displaying them in an app.

Parameters for v1 - Turbidity, Temperature, Photos accross intervals.

Tech Stack - Firebase, Cloudinary, ESP 32 CAM,... more to come.

Current Status - Firebase Setup Completed - Hardware on the way

The Concept -

Providing identity to rivers and waterbodies which are not famous, essential but still overlooked. 
Right now, the scope is limited to local ponds, but it is projected to include rivers too, in the future in order to produce a comprehensive dataset of local waterbodies in order to assist in comprehensive and strategic water resource management.
further, it will include other metrics according to the needs of the particular sites and analysis requirements, but rn we are starting off the v1 with minimal equipment to produce quality work. 
Data will be available publicly through a web/mobile app, whenever appropriate.
